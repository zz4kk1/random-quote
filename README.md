# Random Quote Generator

**Автор:** Иванов Иван

## Описание

Простое приложение для генерации случайных цитат с возможностью просмотра истории и фильтрации по автору или теме. История сохраняется в файл JSON.

## Инструкция по запуску

1. Установите Python 3.x.
2. Запустите `main.py`:
   ```bash
   python main.py
   ```
3. Нажмите «Сгенерировать цитату».
4. Используйте фильтры для поиска в истории.
5. История сохраняется автоматически.
6. База предопределённых цитат находится в `quotes.json`.
7. История хранится в `history.json`.
